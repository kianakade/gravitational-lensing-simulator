# Gravitational Lensing Simulator

Interactive gravitational lensing simulator using PyAutoLens.

## Features
- Interactive lens parameter controls
- Real-time visualization
- RGB image lensing
- Critical curves & caustics

## Installation
```bash
git clone https://github.com/YOUR-USERNAME/gravitational-lensing-simulator.git
cd gravitational-lensing-simulator
pip install -r requirements.txt
python app.py
```

Visit: http://localhost:5002
